今天在量子位看到一篇报道，说 Anthropic 刚发布 Claude Managed Agents 就被开源项目「平替」了，这个开源项目就是 Multica。刚好上周开始用这个项目，就小八了一下。

<!-- more -->

## 报道的说法比较标题党

量子位报道的说法是，Multica 是「开源秒跟」，这个表述并不完全正确。根据项目仓库的信息：

- Multica **今年2月** 就正式创立了
- Claude Managed Agents 是 **今年4月** 才宣布的

所以准确地说，Multica 反而是更早出发的那个。4月的 star 暴涨确实是在 Claude Managed Agents 发布之后。

## 一些小观察

1. Multica项目本身也是用 Claude Code 辅助开发的，不知道 Anthropic 作何感想
2. 创始人之前还创办了面向开发者的 AI 搜索引擎 Devv.ai
3. 目前 13.2k stars，1.6k forks，在增长也是相当快

## 怎么看待

我觉得与其说 Multica 是「平替」，不如说它代表了 **一种需求趋势** —— 大家都希望能更系统地使用 AI Agent，就像我也是想要个能通过看板给 Agent 派活的工具，然后找到了 Multica。Claude Managed Agents 应该也是看到了这种需求才推出的。

媒体总是为了点击量，啥都敢写，作为使用者，不用太管媒体说了啥，自己用用才是真的。目前 Multica 正在开发中，相比上周，已经又多了一整个自动化模块了，版本也来到了 0.2.0。期待它成为能新时代的项目管理工具！
