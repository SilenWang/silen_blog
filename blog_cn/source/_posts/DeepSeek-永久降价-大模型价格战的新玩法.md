最近 AI 圈子最热闹的事，莫过于各家大模型厂商集体涨价。OpenAI 涨了，Anthropic 涨了，连 Google 的 Gemini 也跟着调价。就在大家默默接受"AI 越来越贵"这个事实的时候，DeepSeek 干了件离谱的事——V4-Pro 永久降价，直接降到原价的四分之一。

<!-- more -->

## 逆势而为的 DeepSeek

其实 DeepSeek V4 发布那会儿的定价就已经够惊人了。V4-Flash 缓存命中只要 0.02 元/百万 tokens，这个价格说实话，比很多个人搭的推理服务还便宜。V4-Pro 首发 2.5 折，当时大家以为只是首发优惠，完了就恢复原价。

结果呢？人家在官网公告里写得很清楚：**2026 年 5 月 31 日优惠结束后，V4-Pro 永久调整为原价的 1/4**。这不是什么"限时特惠"，这是永久降价。

对比之下，隔壁 Anthropic 的 Opus 4.6 价格不降反升，Sonnet 4.5 也维持高位。OpenAI 的 GPT-5 系列定价更是一路走高。在这个大环境下，DeepSeek 的操作显得格外扎眼。

## 他们到底怎么做到的？

说实话，这个问题我思考了很久。从商业逻辑上看，大模型推理的成本主要来自计算资源，而英伟达的 GPU 可不便宜。所有厂商都面临着同样的成本压力，凭什么 DeepSeek 就能把价格打下来？

从技术角度看，DeepSeek V4 引入了一些有意思的创新。他们的 DSA（DeepSeek Sparse Attention）稀疏注意力机制在 token 维度上做了压缩，理论上确实能降低计算和显存需求。V4-Flash 作为小参数版本，激活参数更少，推理成本更低，卖便宜点也说得过去。

但问题在于：**这也便宜得太多了**。V4-Pro 永久降到 1/4，几乎是在说他们之前的定价有 75% 的利润空间？还是说他们的实际推理成本已经低到了难以想象的程度？

有一种可能的解释是：DeepSeek 在赌规模效应。通过极低的价格吸引大量用户，把 API 调用量做上去，然后用规模摊薄固定成本。这在互联网行业不是什么新鲜事——烧钱换市场，然后再想盈利的事。

但 AI 推理和传统互联网服务不一样。每一次 API 调用都是实打实的计算开销，边际成本并不会因为用户多了就无限趋近于零。如果调用量暴增，他们要么在算力上投入更多，要么服务质量下降。目前看 V4 系列确实没有出现大规模排队问题，说明他们的算力储备可能比外界想象的要充足。

## 这对市场意味着什么？

对开发者来说，这当然是好事。DeepSeek 的 V4-Flash 现在已经是我主力使用的模型，成本低到几乎可以忽略不计，写代码、做翻译、改文案，根本不用心疼 token 消耗。V4-Pro 虽然贵一点，但按永久降价后的价格看，对于复杂任务来说依然很有竞争力。

但从行业角度看，DeepSeek 这种做法其实是在倒逼整个市场洗牌。如果其他厂商不跟进降价，用户会持续流失到 DeepSeek。如果跟进降价，利润空间被压缩，中小厂商可能直接出局。到最后，大模型 API 市场可能变成一场"谁能把成本控制到极致"的竞赛。

## 一些担忧

不过我也有些隐忧。价格战这种东西，短期看消费者获益，长期看未必。如果 DeepSeek 因为定价过低导致盈利能力不足，后续的模型研发投入会不会受影响？V4 确实很强，但 R1 到 V4 中间也经历了不少波折，研发投入不是小数目。

另外，一家独大对生态也不是好事。目前 DeepSeek 在国内确实是独一档的存在，但如果其他厂商都被价格战打垮了，以后的选择空间反而变小了。

当然，作为一个普通用户，我现在的态度很简单：趁便宜，多用。技术进步带来的红利，能享受的时候就好好享受吧。
